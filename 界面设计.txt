界面设计需求文档：AutoLedger AI（萌系记账风格）

一、全局视觉规范

1.1 色彩体系

• 主色 Primary：#F8BBD0 樱花粉，用于底部导航选中态、FAB、关键按钮、进度条填充

• 辅色 Secondary：#B2EBF2 薄荷蓝，用于分类图标背景、卡片辅助底色

• 辅色 Tertiary：#FFF9C4 鹅黄，用于 AI 建议气泡、高亮标签

• 支出色 Expense：#FF8A80 蜜桃红，用于支出金额、超支标记（禁用正红）

• 收入色 Income：#A5D6A7 薄荷绿，用于收入金额

• 背景 BG：#FFF5F7 暖粉白，全页背景

• 卡片 Card：#FFFFFE 近白，Surface 底色，elevation=2dp

• 文字 Primary：#5D4037 深咖，标题和大数字，禁用纯黑

• 文字 Secondary：#8D6E63 浅咖，副标、备注、时间戳

• 分割线：#F0E6E8 淡粉灰

1.2 形状规范

• 页面卡片（概览、列表项）：圆角 24dp

• 按钮（生成报告、保存）：圆角 20dp

• 分类图标容器：圆形，直径 48dp，背景 Primary 12% 透明度

• BottomSheet 顶部拖拽把手：圆角 16dp，宽 40dp 高 4dp

• 底部导航栏：顶部圆角 24dp（悬浮岛状）

• 输入框：圆角 16dp

• AI 对话气泡：圆角 20dp

1.3 字体规范

• 首页结余大数：36sp，Bold，深咖

• 卡片标题（本月收支）：14sp，Medium，浅咖

• 卡片数字（收入/支出）：20sp，SemiBold，收入绿/支出红

• 列表商户名：16sp，Medium，深咖

• 列表时间/备注：12sp，Regular，浅咖

• 按钮文字：16sp，SemiBold，白色

• 底部导航标签：11sp，Regular，选中 Primary / 未选浅咖

1.4 动效规范

• 页面进入（Navigate）：slideInHorizontally + fadeIn

• 返回上级：slideOutHorizontally + fadeOut

• 首页卡片入场：slideInVertically(offsetY=1/4高度) + fadeIn(300ms)

• FAB 点击：scale 1.0 → 0.88 → 1.0 弹性回弹

• 记账保存成功：Lottie 打勾（500ms）+ 金额数字 countUp 动画

• 分类图标选中：放大 1.0 → 1.15 + 背景色渐变过渡 200ms

• AI 回复加载：三点跳动指示器（Compose 无限过渡动画）

二、底部导航栏

• 容器白色 #FFFFFE，顶部圆角 24dp，底部直角贴手机底边，elevation=8dp

• 高度 64dp（不含系统导航栏）

• 中央预留 56dp 宽圆形缺口区域，用于悬浮 FAB 覆盖

• 四个 Tab：首页（Home 图标）、统计（PieChart 图标）、AI助手（SmartToy 图标）、我的（Person 图标）

• 图标 24dp，文字 11sp；选中樱花粉，未选浅咖；选中态下方加直径 4dp 小圆点指示器

• 中央悬浮 FAB：直径 56dp，樱花粉背景，白色 + 号图标 24dp，elevation=6dp；点击弹出记账 BottomSheet

三、各页面 UI 需求

3.1 首页（HomeScreen）

顶部问候区：
• 状态栏透明，背景色延伸

• 左侧问候语"嗨，今天也想省钱吗？"深咖 16sp Medium，下方小字"9月已记 23 天 🔥"浅咖 12sp

概览卡片：
• 单个大卡片，圆角 24dp，白色背景，margin 16dp

• 上方小字"本月结余"浅咖 14sp，中间大数字"¥ 1,280.50"深咖 36sp Bold 居中

• 下方左右排列"收入 ¥5,000"和"支出 ¥3,720"，中间竖线分隔，收入绿/支出红 14sp

• 底部预算进度条：高度 8dp，圆角 4dp，已完成部分蜜桃红到鹅黄渐变；下方小字"预算使用 64%"浅咖 12sp

今日速览：
• 两个小卡片横向排列，各占一半宽度，圆角 16dp，薄荷蓝背景 20% 透明度

• 左卡片：☕ 餐饮图标 + "¥38.00"；右卡片：🚇 交通图标 + "¥13.00"

最近记录列表：
• 标题栏"最近记录"深咖 16sp Medium，右侧"查看全部 →"浅咖 12sp，点击跳转全部记录列表

• 列表项：左侧圆形分类图标（48dp，Primary 12% 背景），上行商户名深咖 16sp Medium + 金额蜜桃红/薄荷绿 16sp SemiBold，下行时间+分类浅咖 12sp

• 列表项点击跳转 TransactionEditScreen(id)；左滑出现蜜桃红"删除"按钮

• 下拉刷新 SwipeRefresh

• 底部预留 80dp 空白避开 FAB

3.2 统计页（StatsScreen）

顶部切换：
• 月/季/年胶囊 Toggle，选中樱花粉背景白字，切换带动画重绘图表

支出构成区：
• 标题"支出构成"深咖 16sp Medium

• 白色卡片圆角 24dp，左侧 Vico 空心饼图（中心"总支出\n¥3720"），右侧竖向图例（色块+分类名+百分比+金额）

分类排行区：
• 标题"分类排行"深咖 16sp Medium

• 纵向列表，每行：排名序号浅咖 14sp、分类图标圆形 32dp、分类名深咖 14sp Medium、右侧金额深咖 16sp SemiBold + 笔数浅咖 12sp

每日趋势区：
• 标题"每日趋势"深咖 16sp Medium

• Vico LineChart，X 轴日期缩写，Y 轴金额，峰值点标蜜桃红圆点+金额气泡，下方标注"单日最高 09.14 ¥430"

3.3 AI助手页（AiScreen）

顶部栏：
• 标题"✨ 智能建议"深咖 18sp Bold，右侧时钟图标跳转 AiHistoryScreen

对话内容区：
• 可滚动 LazyColumn

• AI 建议气泡：鹅黄背景 #FFF9C4，圆角 20dp，内部文字深咖 14sp，padding 12dp

• 建议卡片：白色卡片圆角 16dp，左侧 4dp Primary 竖线，内部标题深咖 14sp Medium + 正文浅咖 13sp

• 加载中气泡：鹅黄背景内三点跳动动画

• 首次进入无历史时显示欢迎气泡："嗨~ 我是你的 AI 理财小助手，先记几天账，我就知道怎么帮你省钱啦 ✨"

生成报告按钮：
• 全宽樱花粉按钮，圆角 20dp，文字"✨ 生成我的消费报告"白色 16sp SemiBold

• 点击后 loading 转圈，2-3 秒后逐条弹出建议气泡

底部输入框：
• 固定在底部，白色背景圆角 24dp，placeholder"问问 AI..."浅咖 14sp，右侧 Send 图标樱花粉

• 首次云端 API 发送前弹 Dialog 提示"将发送脱敏数据至 AI，是否继续？"

3.4 我的页（SettingsScreen）

顶部区域：
• 渐变卡片（樱花粉到薄荷蓝）圆角 24dp，内部显示"我的记账"深咖 18sp Bold + "本地数据，安全可靠"浅咖 12sp

分组列表（每组间距 12dp）：
• "账号"组：通知监听权限（左侧铃铛图标+标题，右侧绿点●已开启/灰点●未开启，实时检测）；后台保活引导（电池图标，点击弹图文 Dialog）

• "数据"组：分类管理（跳转 CategoryManageScreen）、规则管理（跳转 RulesManageScreen）、账户管理（跳转 AccountsManageScreen）、预算设置（跳转 BudgetScreen）、搜索记录（跳转 SearchScreen）、通知日志（跳转 LogsScreen）

• "AI"组：Toggle 开关（本地规则/云端 API），开启云端时展开 API Key 输入行（密码模式+眼睛图标切换明文）

• "其他"组：清空所有数据（红色文字，点击弹 Dialog 要求输入"确认清空"四字才能确定）；关于（版本号）

列表项统一样式： 白色卡片每行高 56dp，左图标 24dp Primary 色，标题深咖 16sp，右侧箭头或开关

3.5 记账 BottomSheet（全局共用）

触发： 点击底部中央 FAB 弹出，半模态，圆角 28dp

顶部栏： 拖拽把手居中，右侧时间选择器（默认当前时间）

分类选择： 两行各 4 个圆形图标网格，48dp 容器 Primary 12% 背景，选中放大 1.15 + 背景变 Primary 30%，末尾"⋯"展开全部分类

金额显示： 居中大数字 36sp 深咖，默认"0"

数字键盘： 4×4 网格，按键圆角 16dp

可选输入（折叠）： 备注 EditText 圆角 12dp、账户下拉选择

保存按钮： 底部全宽樱花粉按钮圆角 20dp，"保存"白色 16sp SemiBold；点击收起 Sheet + Lottie 打勾 + 首页刷新

3.6 二级页面通用规范

所有二级页面使用 TopAppBar（透明背景，深咖返回箭头+标题），内容区白色卡片圆角 24dp 置于暖粉白背景上。

• 分类管理页：列表展示所有分类，每行彩色圆点+分类名+编辑/删除图标；底部"添加分类"按钮弹 Dialog

• 规则管理页：列表展示关键词→分类映射，每行关键词+分类名+编辑/删除；底部"添加规则"弹 Dialog

• 账户管理页：列表展示账户，每行图标+账户名+余额；底部"添加账户"弹 Dialog

• 交易编辑/详情页：接收 ID 参数，表单含金额、分类下拉、账户下拉、时间选择器、备注、来源标签（只读）；底部"保存"和红色"删除"

• 预算设置页：月度总预算输入 + 分类预算列表，每项可设单独限额

• 搜索页：顶部搜索栏+筛选按钮，下方结果列表（同首页样式），支持商户名/备注/分类/时间范围筛选

• 日志查看页：列表展示通知抓取日志，每条时间+来源+原始文本+解析结果（失败项标蜜桃红）

• AI 历史建议页：按时间倒序列表展示历史报告，每条卡片日期+摘要，点击展开完整内容

四、交互规范汇总

• 所有页面返回手势统一 NavHost popBackStack

• 列表均为 LazyColumn

• 删除操作均需二次确认 Dialog

• 金额统一保留两位小数，千位分隔符

• 深色模式暂不支持
