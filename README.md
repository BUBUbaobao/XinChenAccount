# 昕辰记账

个人本地记账微信小程序 — 手动记账 + 粘贴识别 + 统计图表 + 预算管理 + AI 智能分析。

## 技术栈

微信小程序原生框架（WXML + WXSS + JS）| Canvas 2D 图表 | DeepSeek AI 流式对话 | wx.Storage 本地存储

## 功能模块

### 记账

- **手动记账**：FAB 悬浮按钮 → 分类网格 + 数字键盘 + 备注/账户/时间，绿底白字保存
- **粘贴识别**：复制微信支付消息 → 打开小程序自动弹出确认横幅 → 一键记账
- **编辑/删除**：点击记录进编辑页，左滑（QQ 风格跟手拖拽+弹性吸边）删除

### 首页

- 月度概览卡片（收入/支出/结余）
- 今日速览（Top2 支出分类 + 其他汇总）
- 最近记录列表（分页 20 条/页，下拉刷新）
- 预算进度条 + 超支提醒通知条
- 粘贴识别确认横幅

### 统计

- **环形图**：Canvas 2D 空心甜甜圈，中心显示总金额，10 色马卡龙调色盘
- **分类排行**：金额降序列表（排名+颜色点+emoji+金额+笔数+百分比进度条）
- **柱状趋势图**：纯 View 实现，峰值柱深色高亮+金额标注
- 月/季/年视图切换 + 支出/收入切换
- 泛化计算函数（一次支持三种时间维度）

### 规则管理

- 60 条预置关键词映射规则（餐饮/交通/购物/娱乐/医疗/教育/居住/服饰/通讯）
- 自定义规则 CRUD + Switch 启用/禁用开关
- 预置规则不可删除
- 规则即时生效于粘贴识别

### 搜索

- 关键词搜索（备注/分类/账户）
- 组合筛选：类型胶囊 + 分类 Picker + 日期范围 + 金额范围
- 结果分页加载，复用首页列表样式
- 空状态「🔍 未找到匹配记录」

### 预算管理

- 总预算 + 各分类月度预算设置，底部弹窗修改，设 0 即删除
- 首页蜜桃红→鹅黄渐变进度条
- 超支蜜桃红左边框通知条
- 月份垂直隔离（切换月份进度归零）

### CSV 导入导出

- **导出**：UTF-8 BOM + 7 列表头，`wx.shareFileMessage` 分享到聊天，失败降级复制剪贴板
- **导入**：`wx.chooseMessageFile` 选择文件 → 逐行解析（支持引号转义）→ 分类名/账户名不区分大小写智能匹配 → 去重 → 批量写入
- 导入结果弹窗（成功 N 条，跳过 M 条）

### AI 智能分析

- **双轨架构**：本地规则引擎 + 云端 DeepSeek
- **本地引擎**：超支检测 + 高频消费识别 + 大额异常检测 + 问题感知定向回答（排行/省钱/预算/结余）
- **云端模式**：DeepSeek API 流式输出（SSE 解析 + UTF-8 解码 + `enableChunked`）+ Markdown 实时渲染
- **API Key**：纯 JS Base64+XOR 双重加密存储，密码模式输入 + 👁 切换明文
- **对话**：纯内存存储（切 Tab 不丢），用户白底右对齐 / AI 鹅黄底左对齐气泡，三点加载动画
- **报告生成**：一键生成月度消费报告

### 设置

- 顶部渐变卡片（总记录数 + 存储用量）
- 使用指引弹窗（三步操作说明）
- 分类/规则/账户/预算管理入口
- CSV 导入导出入口
- AI 模式切换 + API Key 配置
- 清空数据（输入"确认清空"二次确认）
- 关于弹窗（版本号+技术栈）
- Storage ≥80% 警告提示

### 解析日志

- parser 自动记录每次粘贴识别结果
- 日志列表：成功✅/失败❌ 标签 + 原文 + 解析结果（保留最近 100 条）

## 性能优化

- Storage 内存缓存层 + 脏标记（dataVersion），零 I/O 读取
- `getHomeData()` 单次遍历合并 4 项计算
- 泛化计算函数（月/季/年统一接口）
- 首页卡片 slideUp 入场动画
- 无 ECharts 依赖（Canvas 2D + View 图表）

## 使用方式

1. 微信开发者工具导入项目
2. 详情 → 本地设置 → 勾选「不校验合法域名」
3. 编译运行
4. AI 云端模式：设置 → AI 分组 → 填入 DeepSeek API Key（sk-开头）

## 数据存储

所有数据 100% 存储于设备本地 `wx.Storage`，无需网络即可使用核心记账功能。
